# プライバシーポリシー

**レシメモ**（以下「本アプリ」といいます）は、個人開発者（以下「開発者」といいます）が提供する支出管理アプリです。本アプリにおけるユーザーの情報の取り扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます）を定めます。

本アプリをご利用いただく前に、必ず本ポリシーをお読みください。本アプリをご利用いただいた場合、本ポリシーに同意いただいたものとみなします。

## 1. 収集する情報

本アプリは、サーバーへの会員登録などを必要としないアプリです。開発者が本アプリの外部（サーバー等）で収集する情報は、原則としてありません。

ただし、以下の場合に限り、情報が本アプリの外部（外部サービス）に送信されます。

- **レシートOCR（文字読み取り）機能を使った場合のみ**、撮影・選択したレシート画像がテキスト読み取りのため Google Cloud Vision API に送信されます。

上記以外の操作（支出の記録、グラフ表示、設定変更など）では、情報が本アプリの外部に送信されることはありません。

## 2. 収集しない情報

本アプリは、以下の情報を一切収集しません。

- 氏名・住所・電話番号などの登録を必要とする個人情報
- 位置情報
- 端末の連絡先・写真ライブラリ全体へのアクセス（写真は、ユーザーがレシート添付のために選択したものだけを扱います）
- 広告identifier（広告表示機能自体が現時点で存在しないため）

## 3. 銀行口座・クレジットカード情報について

本アプリは、銀行口座・クレジットカードなどの金融機関との連携を一切行いません。口座番号、暗証番号、カード番号、金融機関のログインID・パスワードなど、金融情報を収集・保存することはありません。支出の金額はすべてユーザーご自身が手入力（または後述のOCR読み取り結果の確認・修正）によって記録します。

## 4. 支出データの扱い

金額・日付・カテゴリ・メモなどの支出データは、端末内のローカルストレージ（`AsyncStorage`）にのみ保存されます。開発者のサーバーや第三者のサーバーに支出データが送信されることはありません。

## 5. レシート画像の扱い

- レシート画像は、ユーザーがカメラで撮影した、または写真ライブラリから選択したものだけが本アプリに保存されます。ユーザーが選んでいない写真を本アプリが自動的に収集することはありません。
- 保存されたレシート画像は端末内（アプリ専用の保存領域）にのみ保持されます。
- OCR機能を使わない限り、レシート画像が外部に送信されることはありません。OCR機能を使った場合の扱いは「1. 収集する情報」および「6. OCRテキストの扱い」をご確認ください。

## 6. OCRテキストの扱い

- OCR機能を使うと、レシート画像が文字読み取りのために Google Cloud Vision API に送信され、読み取り結果のテキストが返されます。
- 読み取られたテキストおよびそこから抽出した金額・日付・店名などの候補は、端末内にのみ保存されます。開発者がこれらの内容を閲覧・収集することはありません。
- OCRの読み取り結果は、必ずユーザー自身が画面上で確認・修正したうえで記録に反映される仕組みになっています。

## 7. 通知機能の扱い

本アプリの通知（毎日のリマインダー、予算超過のお知らせなど）は、すべて端末内でスケジュールされるローカル通知です。外部のプッシュ通知サーバーは利用しておらず、通知のために情報が外部に送信されることはありません。

## 8. CSV出力の扱い

支出データをCSVファイルとして書き出す機能は、書き出したファイルを端末内に保存する、またはユーザーが選んだ共有先（メール、クラウドストレージアプリなど）に共有するための機能です。書き出し先の選択・管理はすべてユーザー自身の責任で行っていただきます。開発者がCSVファイルの内容を収集することはありません。

## 9. バックアップ・復元ファイルの扱い

支出・予算データをバックアップするためのJSONファイルも、CSV出力と同様に端末内保存またはユーザーが選んだ共有先への共有によって取り扱われます。バックアップファイルの保管・管理はユーザー自身の責任で行っていただきます。開発者がバックアップファイルの内容を収集することはありません。

## 10. 広告・解析ツールの利用について

本アプリは、現時点で広告表示機能および第三者製の解析（アクセス解析・行動分析）ツールを組み込んでいません。そのため、広告配信や解析のためにユーザーの情報が第三者に送信されることはありません。将来これらを導入する場合は、事前に本ポリシーを更新し、収集する情報の内容を明示します。

## 11. 第三者提供について

開発者は、本アプリを通じて取得した情報を、第三者に提供・販売することはありません。唯一の例外は、「1. 収集する情報」に記載のとおり、OCR機能を使った場合にレシート画像の読み取りのためGoogle Cloud Vision APIに送信される場合です（この送信は文字読み取りという処理目的のみに利用され、Googleによる取り扱いは各サービスのプライバシーポリシーに従います）。

## 12. データの保存場所

支出データ・レシート画像・OCRテキスト・設定などは、いずれもユーザーの端末内にのみ保存されます。開発者が管理するサーバーにユーザーのデータを保存する仕組みは、現時点ではありません。

## 13. ユーザーによるデータ削除方法

- 個別の支出データは、支出の詳細画面から削除できます。
- すべてのデータをまとめて削除したい場合は、設定画面の「全データ削除」機能をご利用いただけます。
- アプリ自体をアンインストールすることでも、端末内に保存されたすべてのデータが削除されます。

## 14. 子どもの利用について

本アプリは、特定の年齢層を対象とした年齢確認機能を設けていません。本アプリは会員登録や個人情報の入力を必要としない設計になっていますが、保護者の方は、お子様が本アプリをご利用になる場合、本ポリシーの内容をご確認いただくことを推奨します。

## 15. 今後の機能追加とポリシーの更新について

本アプリでは、以下の機能を今後追加する可能性があります。これらを実装する際は、実装前に本ポリシーの内容を見直し、収集する情報の範囲・保存場所・第三者提供の有無などを明記したうえで更新します。

- クラウド同期・複数端末間のデータ共有（Appleでサインインなどを利用したアカウント連携）

本ポリシーの内容に重要な変更がある場合は、本アプリ内および本ドキュメントの改定日を更新することでお知らせします。
