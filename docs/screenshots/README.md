# スクリーンショット配置場所

ルートの `README.md` にスクリーンショットを表示するため、このフォルダに以下のファイル名で画像を配置してください。

| ファイル名 | 内容 |
|---|---|
| `home.png` | ホーム画面（今日/今月の合計・カテゴリ別内訳・最近の支出一覧） |
| `add-expense.png` | 支出追加画面（金額・日付・カテゴリ・メモ・レシート写真の入力） |
| `graph.png` | グラフ画面（カテゴリ別支出の円グラフ） |

形式はPNG推奨（他形式を使う場合はルートの`README.md`側の拡張子も合わせて変更してください）。
