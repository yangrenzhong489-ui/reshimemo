# スクリーンショット配置場所

ルートの `README.md` にスクリーンショットを表示するため、このフォルダに以下のファイル名で画像を配置してください。

| ファイル名 | 内容 |
|---|---|
| `home.png` | ホーム画面（今日/今月の合計・カテゴリ別内訳・最近の支出一覧） |
| `add-expense.png` | 支出追加画面（金額・日付・カテゴリ・メモ・レシート写真の入力） |
| `graph.png` | グラフ画面（カテゴリ別支出の円グラフ） |
| `monthly-report.png` | 月別レポート画面（今月と前月の比較・カテゴリ別ランキング） |
| `settings.png` | 設定画面（通知・バックアップ・CSV出力などの設定項目） |
| `plans.png` | プラン画面（Free / Plus / Pro の機能比較） |

形式はPNG推奨（他形式を使う場合はルートの`README.md`側の拡張子も合わせて変更してください）。

## ⚠️ 撮影時の注意（個人情報・実レシート情報）

- 実際に買い物したレシートや自分の支出履歴をそのまま撮影に使わないでください。店名・購入品・金額・場所などの個人情報が写り込みます。
- **必ずサンプルデータ（ダミーの支出・ダミーのレシート画像）を入力した状態で撮影してください。**
- 詳しい撮影チェックリストはルートの `README.md` の「📸 スクリーンショット」セクションを参照してください。
